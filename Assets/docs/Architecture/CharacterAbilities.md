Las acciones especiales del personajes



Cada ability es un módulo independiente



Ejemplo

&nbsp;-JumpAbility

&nbsp;- DashAbility

&nbsp;- SlideAbility

&nbsp;- VaultAbility

&nbsp;- WallRunAbility



Responsabilidades 

&nbsp;- Decidir si puede activarse

&nbsp;- Modificar el motor

&nbsp;- Finalizar cuando corresponde



Las abilities no leen input directamente
