Es la capa responsable del movimiento físico del personaje



Reponsabilidades: 
	- Mover el personaje usando CharacterController

&nbsp;	- Aplicar gravedad

&nbsp;	- Mantener velocidad actual

&nbsp;	- Recibir impulsos de abilities



Funciones que expondrá: 



&nbsp;	- SetPlanarVelocity()

&nbsp;	- SetVerticalVelocity()

&nbsp;	- AddImpulse()

&nbsp;	- Move()



Regla fundamental: Ningún sistema mueve el personaje excepto el motor
